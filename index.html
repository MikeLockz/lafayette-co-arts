Public Art in Lafayette, CO USA

This repository contains data on public art in Lafayette, CO USA. Included are Alley Art Murals, Sculptures, and Permanent Collection of the City. The data is licensed under the Apache Open Source License and is free to use and distribute.

Download the data in CSV for JSON format

If you would like to contribute to or edit the database of public art in Lafayette, CO you have two options:

  1. Fork our github repository
  2. Or, download the CSV file, edit it in excel, and email changes to Jon Saints saintsjd@gmail.com
